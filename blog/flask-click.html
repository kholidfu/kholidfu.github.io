1 Flask dan Flask-Click

Semenjak versi 0.11, Flask menyertakan fitur baru yakni click command line interface. click sendiri merupakan pustaka python yang dikembangkan oleh pembuat Flask.

Dengan fitur ini, membuat custom command di Flask menjadi sangat mudah.

Berikut ini adalah cara penggunaannya: 

import click

@app.cli.command()
def initdb():
    '''Init db.

    '''
    click.echo('Init the db')

Sebelumnya, pastikan path menuju ke aplikasi Flask Anda sudah terkonfigurasi. Saya prefer dengan menambahkan ke berkas bin/activate: 

export FLASK_APP=$VIRTUAL_ENV/app/__init__.py
export FLASK_DEBUG=1

Kemudian silakan restart terminal atau ketik perintah source ~/.bash_profile, dan jalankan perintah initdb diatas: 

flask initdb

Secara default, Flask sudah menyertakan perintah flask run untuk menjalankan localhost server. Tapi Anda dapat melihat daftar parameter dengan menggunakan perintah 

flask --help
